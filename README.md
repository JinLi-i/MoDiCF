# MoDiCF

This is the repository for the submission 1406, "Generating with Fairness: A Modality-Diffused Counterfactual Framework for Incomplete Multimodal Recommendations".

## Requirements
- Python 3.9
- PyTorch 2.3.1
- Numpy 1.26.3
- Scipy 1.13.1
- SciKit-Learn 1.5.0
- Wandb 0.17.3

## Data
Please download the datasets from the following link:
- https://github.com/HKUDS/MMSSL

and place them in the `data` folder.

## Running the code
To run the code, execute the following command:
```bash
python main_baby.py
```
```bash
python main_tiktok.py
```
```bash
python main_allrecipes.py
```
